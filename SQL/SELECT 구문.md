# SQL

- SQL은 테이블을 검색해서 데이터를 검색하거나 갱신할 때 사용하는 언어

# SELECT

- 데이터베이스를 이용하는 데 핵심이 되는 처리 : 검색
- 질의(query) 또는 추출(retrieve) 라고 부름
- SELECT 구 FROM [테이블 이름]
- WHERE 구
	- 특정 조건에 맞는 레코드만 추가적인 조건 지정
	- 실제 사용하는 조건은 복합 조건
	- IN으로 묶어서 OR 조건을 간략하게 작성
	- NULL 레코드를 선택할 때는 'IS NULL'이라는 키워드 사용
	- 반대로 NULL이 아닌 레코드를 선택할 때는 'IS NOT NULL'이라는 키워드 사용
- GROUP BY 구
	- 테이블에서 데이터 선택 뿐 아니라 합계, 평균 등 집계 연산 가능
- HAVING 구
	- 선택된 결과 `집합`에서 또다시 조건을 지정 
- ORDER BY 구
	- SELECT 구문의 결과 순서를 보장하려면 명시적으로 순서를 지정해야 함
	- 오름차순(ASC), 내림차순(DESC)
